## C vs C++

* C
```cpp
#include <stdio.h>
int main() {
	printf("Hello, World!\n");
	return 0;
}
```

* C++
```cpp
#include <iostream>
using namespace std;
int main() {
	cout <<"Hello World"<<endl;
	return 0;
}
```
