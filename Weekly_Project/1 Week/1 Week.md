### C vs C++

* C
```c
#include <stdio.h>
int main() {
	printf("Hello, World!\n");
	return 0;
}
```

* C++
```c++
#include <iostream>
using namespace std;
int main() {
	cout <<"Hello World"<<endl;
	return 0;
}
```

### C++ 출력 연습
